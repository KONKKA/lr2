Тема роботи

Інформаційна система автоматизованого керування проксі-серверами для комерційного використання

Information System for Automated Management of Proxy Servers for Commercial Use

Короткий опис

Telegram-бот-магазин для оренди проксі. Система автоматично розгортає проксі на відповідному сервері, при оренді та відразу видає клієнту дані для доступу.

Мета роботи

Створити Telegram-бот-магазин для оренди проксі, з автоматичним розгортанням проксі на сервері.
